Rod Weight Calculator

A simple Python calculator to find the volume and weight of a cylindrical rod using diameter, length, and material density.

This small tool was built while learning Python to automate basic engineering calculations.

Features

Calculates volume of a cylindrical rod

Calculates weight in grams and kilograms

Simple console input

Useful for quick engineering estimation
